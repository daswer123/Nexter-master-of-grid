Комманды сборки:
1.  npm run build — Запускает режим сборки, создает webP изображения, минифицирует код, группирует медиа запросы
2.  npm run dev — Запускает режим разработчика
3. npm run deploy — Создает архив с сайтом в режиме сборки
4. npm run makeFontsFile — Преобразовывает все шрифты из формата otf, ttf в woff и woff2 , автоматически создает файл fonts.scss ( важно при сборки этого файла не должно существовать )
5.npm run svg — создает спрайт из всех svg картинок в папке svgicons
